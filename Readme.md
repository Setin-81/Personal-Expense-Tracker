Personal Expense Tracker (C++)

--------

A simple console-based personal expense tracker written in C++. This application allows users to add, view, calculate total, and save their personal expenses using a CSV file. Perfect for beginners and those who want to showcase basic C++ skills including OOP, file handling, and user input validation.

Features

Add Expense: Record a new expense with date, category, amount, and description.

View Expenses: Display all recorded expenses in a neatly formatted table.

Total Expenses: Calculate the total amount of all expenses.

Save & Load Data: Persist data to a CSV file (expenses.csv) so it can be retrieved later.

User-friendly input validation: Prevents crashes on invalid input.

--------

Usage

Run the application.

Choose an option from the menu:

1 Add a new expense

2 View all expenses

3 Calculate total expenses

4 Save & Exit

Data is automatically saved to expenses.csv.
